Music Space was created for music lovers that can search for any artist and get information on their top hits and albums. On top of that, any upcoming shows that are available on Ticketmaster will also display. But what if you cannot make it to that event? Click the buy tickets button to be taken to Ticketmaster and see all of their upcoming shows that may be closer to your area. All of this is built using 2 different API's to gather and render this data based on the search parameters. HTML5 and Bulma/CSS3 were used to create the front end and JavaScript was used to render the information and fetch the data.

Recipedia is a heavy database driven app. It allows users to create and look up recipes within the database via keyword search. The ingredients table is stored separate from the recipe data through relational ID's. The backend is ran through Node Express and the database was built using Sequalize.
